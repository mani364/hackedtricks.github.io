Payment Collection (2019)

This was an android application developed for the finance department of Pinnacle Smart Technologies, LLC Dubai. This application was designed to be used by the admin staff and drivers. The main purpose of this application was to assign the payment collection tasks to drivers and the assigned tasks can also be monitored by the admin to check weather if the status is "completed" or "not completed".

UiPath Automation (2019)

  • Developed a Robotic Process Automation program using UIPath at Pinnacle Smart Technologies LLC, Dubai which automatically logs into sever database->queries the database->Converts the result into pdf format -> Downloads the result -> sends an email to management with attached report -> Deletes the file after sending email.

  • Developed an automation which will browse www.amazon.ae ->searches for a particular laptop ->Scraps the result and stores in a tabluar format in a CSV file and saves it in a particular location ->Sends an email to the requesting user with the attached CSV file."

  • Also developed an automation which automatically fills out google form supplied with information of 15 to 25 people.
